##初识PHP
作为一个前端，学习后台是非常有必要的
http://images2017.cnblogs.com/blog/1275382/201711/1275382-20171108195520466-1695347651.jpg全世界最好的语言是什么......php，额，这样火的后台语言是每一个小前端都应该去学习的，对，学习，学习使我快乐，我爱学习………..

为什么要学习PHP？

作为一枚刚入坑的前端菜鸟，技术还是停留在静态页面的制作上，静态页面固定，没有和服务器的交互，也没有数据库的支持，实现不了动态的数据更新，所以…….我们要去访问后台，后台，对，那么，PHP语言就可以帮助你去访问后台数据，制作出动态的页面，实时的展示数据，赋予网页全新的的生命，当别人问你是做什么的，那么我们就不再是一个做网页的了，而是一个…会PHP的做网页的……..是不是瞬间高大上了，也不用去麻烦后台小哥哥们了，让你从此，做个高冷的前端小姐姐>_<

进入主题—–我们的口号是：学习，学习，学习…

1、php的书写

<?php ?> 这是标准的PHP语言书写

<? ?> php短标签，这种写法不提倡，取决于服务器配置

2、php的变量

注：php是一门弱类型的语言，定义变量的时候不需要指定变量类型，并且在变量执行过程中可以被赋值成任意类型

（1）数字类型

$number2=-1;

$number3=0123;

$number4=0x123；

(2)字符串类型

复制代码

1 $name=”huahua”;

2 $string1=”string $name”;//在双引号里面变量会被解析 —-string

huahua

3 $string2=’string $name’;//在单引号里变量不会被解析 —-string

$name

4 $string3 = <<<EOF // <<<叫做定界符，这也是声明字符串的一种方式

5 string $name

6 EOF;

.运算符是连接符，相当于js里面的+

1 $name=”luxp”;

2 $welcome=”欢迎”.$name.”来这里”;

3 echo $welcome; //欢迎luxp来这里

当字符串里面包含了双引号或者单引号时需要转义

1 $name.=”欢迎\”$name\”来这里”; 2 echo $name; //欢迎”luxp”来这里

3、数组的应用

打印的结果
Array ( [0] => 数组1
　　　　[1] => 数组2

　　　　[no2] => 关联数组2

注：1、这里的关联数组其实和数字是一样的道理，只是给的键值不一样，可以任意指定键值
2、当[]为空时，就把值放进下标最大的空的数组里

二维数组 数组的元素还是数组
$array1[‘no1’]=array(1,2,3,4);
print_r($array1);
打印的结果

Array ( [0] => 数组1
　　　　[1] => 数组2
　　　　[no1] => Array ( [0] => 1 [1] => 2 [2] => 3 [3] => 4 )
　　　　[no2] => 关联数组2
　　　　[2] => 数组3
　　　　[3] => 数组4
　　　　[6] => 数组6 )

注：这里给关联数组里面又放了一个数组，可以以下面的方式取到数组里面的值
//输出no1的第个成员
echo $array1[‘no1’][2];//3

4、数据之间的转化

当1个字符串和数字类型的相加，会转化为数字类型

1 $num=”abc”;
2 $num=$num+100;
3 echo $num;//100

字符串的开头是数字时可以相加 $num=”abc10”这样就不可以

1 $num=”10abc”;
2 $num=$num+100;
3 echo $num;//110
